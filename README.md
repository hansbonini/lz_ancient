# LZANCIENT
LZ + RLE Compression found in games developed by Ancient

## List of Games
- [SMD] Beyond Oasis
- [SMD] Streets of Rage 2

## Example
Decompress:
```python main.py D "Beyond Oasis (U) [!].gen" font.bin 0x16943c```

Compress:
```python main.py C "Beyond Oasis (U) [!].gen" font.bin 0x16943c```
